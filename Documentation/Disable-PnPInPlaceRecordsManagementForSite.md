# Disable-PnPInPlaceRecordsManagementForSite

## SYNOPSIS
Disables in place records management for a site.

## EXAMPLES

### ------------------EXAMPLE 1------------------
```powershell
PS:> Disable-PnPInPlaceRecordsManagementForSite
```

The in place records management feature will be disabled
