# Get-PnPSite

## SYNOPSIS
Returns the current site collection from the context.

## SYNTAX 

### 
```powershell
Get-PnPSite [-Includes <String[]>]
```

## PARAMETERS

### -Includes
Specify properties to include when retrieving objects from the server.

```yaml
Type: String[]
Parameter Sets: 

Required: False
Position: 0
Accept pipeline input: False
```
